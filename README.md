A JarJar library for Minecraft mods.
```gradle
repositories {
    maven { url 'https://maven.blamejared.com' }
}

dependencies {
    implementation "com.hollingsworth.nuggets:nuggets-[MC_VERSION]:[NUGGETS_VERSION]"
}
```
